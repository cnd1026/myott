# APS Public Notice

MyOTT는 내부 AI Project System(APS)을 기반으로 개발됩니다.

APS는 Sprint, Task, Codex Mode, Review, 문서 기록, AI 협업 방식을 일관되게 운영하기 위한 내부 프로젝트 운영체계입니다. MyOTT에서는 이 운영체계를 사용해 긴 작업 흐름을 유지하고, 새로운 AI PM이나 Codex 스레드가 현재 상태를 빠르게 이어받을 수 있게 합니다.

## 공개 저장소의 APS 범위

이 공개 저장소에는 MyOTT 개발을 이해하고 이어가는 데 필요한 최소한의 Project Memory System 문서만 유지합니다.

공개 저장소에 남기는 문서는 다음 목적을 가집니다.

- MyOTT의 현재 상태를 설명합니다.
- Sprint와 Task 진행 상황을 기록합니다.
- 제품/기술 결정과 개발 규칙을 공유합니다.
- 공개 프로젝트로서 필요한 맥락을 제공합니다.

## Private Repository 관리

APS의 상세 운영 문서와 재사용 가능한 핵심 원칙은 private repository에서 관리할 예정입니다.

Private repository로 이전할 APS 핵심 문서는 MyOTT뿐 아니라 향후 프로젝트와 『AI와 함께 서비스를 만드는 방법』에도 활용될 수 있는 운영체계 자산입니다. 따라서 공개 저장소에는 세부 운영 문서를 직접 노출하기보다, MyOTT에 필요한 최소 참조와 이전 계획만 남깁니다.

## 현재 원칙

- APS 핵심 문서는 삭제하지 않고 이전 준비를 먼저 진행합니다.
- Git history rewrite, force push, filter-repo는 이번 정리 작업에서 사용하지 않습니다.
- 공개 저장소에는 MyOTT 진행에 필요한 문서와 참조만 유지합니다.
- APS 세부 문서의 장기 관리 위치는 private repository로 분리합니다.

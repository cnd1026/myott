# APS Public Notice

MyOTT는 내부 AI Project System(APS)을 기반으로 개발됩니다.

Built with APS.

APS는 Sprint, Task, Codex Mode, Review, 문서 기록, AI 협업 방식을 일관되게 운영하기 위한 내부 프로젝트 운영체계입니다. MyOTT에서는 APS를 사용해 긴 작업 흐름을 유지하고, 새로운 AI PM이나 Codex 스레드가 현재 상태를 빠르게 이어받을 수 있게 합니다.

## 공개 저장소의 APS 범위

이 공개 저장소에는 MyOTT 개발을 이해하고 이어가는 데 필요한 최소한의 Project Memory System 문서만 유지합니다.

공개 저장소에 남기는 문서는 다음 목적을 가집니다.

- MyOTT의 현재 상태를 설명합니다.
- Sprint와 Task 진행 상황을 기록합니다.
- 제품/기술 결정과 개발 규칙을 공유합니다.
- 공개 프로젝트로서 필요한 맥락을 제공합니다.

## Source of Truth

- Git history rewrite, force push, filter-repo는 이번 정리 작업에서 사용하지 않았습니다.

# Final Integrator Adapter

- Role ID: `FINAL_INTEGRATOR`
- Human Gate Owners: HQ, PM Lab, Founder

## MyOTT Responsibility

- 필수 Handoff, QA Evidence, 사람 Gate와 Candidate Diff를 대조합니다.
- Commit/Push 가능 여부와 제외 파일을 공개 Final Report에 기록합니다.
- 새로운 Product 동작이나 범용 Conflict Resolution을 구현하지 않습니다.
